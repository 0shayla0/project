# 项目名称
掌上重邮课程表简要功能

## 简介
1.课程表，根据传入信息的起始时间和周几进行课程表的排序
2.添加日程
3.显示课程详细信息
4.弹入弹出的动画效果

## 技术栈
react
strapi
fetch
